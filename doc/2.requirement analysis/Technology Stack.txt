# Technology Stack

## Introduction

The Technology Stack represents the software tools and technologies used to develop the project. This project uses Tableau for data visualization and GitHub for project management and collaboration.

## Technologies Used

### 1. Tableau Desktop

Tableau Desktop is the primary tool used for data visualization. It helps create interactive dashboards, charts, graphs, and stories from the housing dataset.

### 2. Microsoft Excel / CSV

The housing dataset is stored in Excel or CSV format. Tableau imports this dataset for analysis and visualization.

### 3. GitHub

GitHub is used to store the project files, manage versions, and collaborate with team members.

### 4. Tableau Public

Tableau Public is used to publish the completed dashboard online so that users can access and share it using a web browser.

### 5. Microsoft Word

Microsoft Word is used to prepare the project documentation, including the abstract, diagrams, screenshots, and project report.

## Software Requirements

• Windows 10 or Windows 11

• Tableau Desktop

• Tableau Public

• Microsoft Excel

• Microsoft Word

• GitHub

## Hardware Requirements

• Intel Core i3 Processor or above

• 4 GB RAM or higher

• 20 GB Free Storage

• Internet Connection

## Advantages of the Technology Stack

• Easy to use and learn.

• Supports interactive dashboards.

• Enables effective data visualization.

• Simplifies project collaboration using GitHub.

• Provides online dashboard sharing through Tableau Public.

## Conclusion

The selected technology stack is simple, efficient, and suitable for developing a housing market analysis dashboard. These technologies ensure better visualization, collaboration, and project documentation.
