# Dashboard and Story

## Introduction

The Tableau dashboard was developed to visualize housing market trends and property features. It converts raw housing data into interactive charts and graphs, allowing users to explore the dataset easily.

The Tableau Story combines multiple dashboards into a single presentation that explains the complete analysis in a logical sequence.

## Dashboard Overview

The dashboard contains different visualizations to analyze housing market trends. Users can interact with the dashboard using filters and charts.

The dashboard provides information about:

• House Sale Prices

• Location-wise Analysis

• Bedroom Analysis

• Bathroom Analysis

• Living Area Analysis

• House Grade Analysis

• Year-wise Sales Analysis

## Dashboard Features

The dashboard includes the following features:

• Interactive Filters

• Bar Charts

• Line Charts

• Pie Charts

• Scatter Plots

• Maps

• KPI Cards

• Tooltips

These features make data exploration simple and interactive.

## Tableau Story

The Tableau Story presents the dashboards in a sequence to explain the complete housing market analysis.

The story includes:

Story 1:
Introduction to the Housing Dataset

Story 2:
Sale Price Analysis

Story 3:
Location-wise House Price Analysis

Story 4:
House Feature Comparison

Story 5:
Market Trends and Insights

Story 6:
Conclusion

## Key Insights

• House prices vary significantly based on location.

• Houses with more bedrooms and bathrooms generally have higher sale prices.

• Larger living areas are associated with higher property values.

• Interactive filters help users analyze specific property categories.

• Tableau dashboards simplify complex housing data.

## Benefits

• Easy-to-understand visualizations.

• Interactive data exploration.

• Better business insights.

• Faster analysis.

• Improved decision making.

## Conclusion

The Tableau Dashboard and Story successfully transform housing data into meaningful visual insights. The project helps users analyze market trends, compare property features, and make informed real estate decisions through interactive dashboards.
