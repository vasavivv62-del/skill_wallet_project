# Data Preparation and Business Questions with Visuals

## Introduction

Data preparation is one of the most important steps in data analysis. Before creating dashboards, the housing dataset must be cleaned and organized to ensure accurate visualization and analysis.

## Dataset Description

The project uses a housing market dataset that contains information about house sale prices and property features. The dataset includes:

• Sale Price
• Number of Bedrooms
• Number of Bathrooms
• Living Area
• Lot Area
• Number of Floors
• House Condition
• House Grade
• Year Built
• Location Details

## Data Preparation Process

The following steps were performed before creating the Tableau dashboard:

### Step 1: Import Dataset

The housing dataset was imported into Tableau Desktop from a CSV/Excel file.

### Step 2: Data Cleaning

The dataset was cleaned by:
• Removing duplicate records.
• Handling missing values.
• Checking incorrect values.
• Verifying data consistency.

### Step 3: Data Organization

The columns were arranged properly, and appropriate data types such as Number, Date, and String were assigned.

### Step 4: Data Validation

The cleaned dataset was verified to ensure that all records were correct before visualization.

## Business Questions

The dashboard was developed to answer the following business questions:

1. What is the average sale price of houses?

2. Which locations have the highest property prices?

3. How does the number of bedrooms affect house prices?

4. How does the number of bathrooms influence the sale price?

5. Which house grades have the highest prices?

6. What is the relationship between living area and sale price?

7. Which year had the highest number of house sales?

8. What are the overall housing market trends?

## Visualizations Used

The Tableau dashboard includes:

• Bar Charts
• Line Charts
• Pie Charts
• Scatter Plots
• Maps
• Filters
• KPI Cards

These visualizations help users analyze housing market trends more effectively.

## Benefits

• Easy understanding of housing data.
• Better comparison of property features.
• Interactive analysis.
• Faster decision making.
• Accurate visualization.

## Conclusion

The data preparation process ensured that the dataset was clean, accurate, and ready for visualization. The business questions guided the creation of meaningful dashboards that provide valuable insights into housing market trends.
