# Final Report

## Project Title

Visualizing Housing Market Trends: An Analysis of Sale Prices and Features using Tableau

## Project Summary

The Housing Market Analysis project was developed to analyze property sale prices and housing features using Tableau. The project converts raw housing data into meaningful visualizations through interactive dashboards.

The dashboard enables users to explore housing market trends based on various factors such as sale price, location, number of bedrooms, bathrooms, living area, and house condition. Interactive filters and charts help users understand market behavior and compare different properties effectively.

## Objectives

• Analyze housing market trends.

• Create interactive Tableau dashboards.

• Compare housing features.

• Support better decision making.

• Present meaningful visual insights.

## Project Features

• Interactive Dashboard

• Charts and Graphs

• Maps

• Filters

• KPI Cards

• Tableau Story

## Benefits

• Easy visualization of housing data.

• Interactive analysis.

• Better understanding of market trends.

• Time-saving data exploration.

• Improved decision making.

## Conclusion

The project successfully demonstrates how Tableau can be used to analyze housing market data and create interactive dashboards. The dashboards provide valuable insights that help buyers, sellers, investors, and analysts understand housing market trends effectively.
