# Performance Testing

## Introduction

Performance Testing is conducted to evaluate the efficiency, reliability, and responsiveness of the Tableau dashboard. It ensures that the dashboard performs well while handling housing market data and provides accurate visualizations without delays.

## Objectives

The main objectives of performance testing are:

• Verify that the dashboard loads correctly.

• Ensure all charts and graphs display accurate information.

• Test the functionality of filters.

• Check the response time of dashboard interactions.

• Ensure smooth navigation between dashboards and stories.

## Test Cases

### Test Case 1: Dashboard Loading

Description:

The dashboard opens without any errors and displays all charts correctly.

## Performance Results

• Dashboard loading time was satisfactory.

• All filters worked correctly.

## Conclusion

The performance testing confirmed that the Tableau dashboard works efficiently and provides accurate visualizations. All major functionalities performed successfully, making the dashboard reliable for housing market analysis.
